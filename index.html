Abstract

Radiochromic film dosimetry has been a valuable and important quality assurance (QA) tool for high resolution 2D map dose verification in radiotherapy (RT) treatment plans, mainly due to its desirable properties as a dosimeter and the relatively simple setup for irradiation and processing. However, the absence of open source software for film anal- ysis limits its routine use, especially in countries where economic resources to acquire specialized software are tight. Aiming to overcome this limitation, FilmQADose, an open source Python-based software for radiochromic film anal- ysis was developed, allowing source modification to add new utilities. Main functionalities of FilmQADose include: creation of various calibration functions, film digitization filtering and noise reduction, 2D dose map generation with single and multichannel methods and plan-to-film comparison tools using DICOM RT files from the treatment planning system (TPS), including Γ analysis and several visualization options. Software workflow was assesed on single field, pyramidal and breast IMRT treatment plans. The first version of FilmQADose proved to be a useful, reliable and easy- to-use software for radiochromic film analysis allowing accurate and reproducible RT plan verification, thus enhancing accessibility and use of film-QA, especially in developing countries where it can be one of the best alternatives to make radiation treatments secure and effective.
